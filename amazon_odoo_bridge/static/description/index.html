Are you managing one or more stores in Amazon Marketplace and still looking for a centralised management solution for your online stores?

If you are running one or multiple stores in the Amazon marketplace and looking for a data management solution in Odoo then Odoo Amazon Connector perfectly fits your requirements.

Whether it is seller data(like orders and catalogs) or inventory you can manage it all with the help of the module.

Amazon Odoo Bridge facilitates synchronization, import and export as per your business requirements.

You can also connect to multiple e-commerce and marketplaces to your single Odoo instance with the help of Odoo Multi-channel Sale.

Connect Your Odoo With Multiple Amazon Seller Accounts

Odoo Multichannel Amazon Connector perfectly works in such situations. You can simply connect one or more Amazon Seller Accounts to your Odoo; and manage its data in your single Odoo Instance.

The need of managing the data in a centralised way cannot go unnoticed as it helps to track the data efficiently. When data is decentralized and disorganized, it becomes an obstacle rather than an asset.

Importing Catalogs in Odoo

You can import Data like Products, Categories, attributes, as well as attribute values from Amazon to Odoo

The module makes the Import process quick and easy. You can import data manually or via cron scheduler. The data is stored in the form of feeds in Odoo and then automatically creates data on the basis of the settings.

Mapping is created in the module to avoid duplicity of the data.

Exporting Catalogs in Odoo

Odoo Amazon Connector has made the export process really convenient. Now, you can export products along with its category from Odoo to Amazon.

You can export products directly from the specific product form.

Importing Orders in Odoo

You can import Orders along with the data associated with it from Amazon to Odoo. The import function is simplified with the help of module Odoo Multichannel Amazon Connector.

Orders statuses like cancel, shipment and invoice are automatically imported at the same time.

Moreover, data like customers and their addresses, taxes, delivery methods, payment methods, currency is also imported along with it.

Key Features

Below is the detailed list of features from Odoo MultiChannel Amazon Connector

Importing Products in Odoo

  • Products are manually imported in bulk
  • You can also import products via cron scheduler
  • Product details (like category, attributes, attribute values) are imported along with the products

Manually Export Products from Odoo

  • You can manually export products on bulk
  • Or, You can manually export selected products

Importing Orders in Odoo

  • Orders are manually imported in bulk
  • You can import orders via cron scheduler
  • Order details (like customers and their addresses, taxes, delivery methods, payment methods, currency) are imported along with the orders
  • Order statuses (like cancel, shipment and invoice) are also imported along with the orders

Manually Map Orders Statuses

  • You can manually map order statuses from Odoo end
  • Fill the corresponding Order status of Amazon and Odoo in the Order Status table
  • Amazon Order State like-Completed, On-Hold, Refunded etc. you need to configure the Order States of both channels accordingly.
  • You need to configure the Order States of both channels; Order State in Amazon are like-Completed, On-Hold, Refunded etc

Real time Synchronization

  • You can sync Inventory from Odoo to Amazon in real-time.

Managing Synchronization History

  • All the synchronization history is managed in Odoo

Enhanced Dynamic Dashboard

  • Introduced a dynamic dashboard to track the data of Amazon to your Odoo.
  • Import/Export the data of any channel directly from the dashboard.
  • Well-designed Donut Chart and Line Graph to analyze the data.
  • The dashboard also offers Clickable tabs and data cards.

Clickable Data Cards for the Details

  • View the imported Product, Category, Order and Customers in the Tabs
  • The tabs are clickable and direct you to the respective mapping, feeds as well as sync history.

Line Graph to Analyze the data

  • Drop down to Select the Data Type (Order, Product, Category and Customer)
  • Track the data the Line Graph

Donut Chart to Analyze the data

  • Drop down to Select the Data Type (Order and Product)
  • Track the data of connected Amazon instance from the chart
